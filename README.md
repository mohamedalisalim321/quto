# Quto — Minimal Quotes App

A clean, dark-mode Flutter application designed for reading, saving, and discovering inspirational quotes.  
Built with simplicity, performance, and modern UI in mind.

---

## ✨ Features

- 📜 Browse curated quotes
- ⭐ Save favorites
- 🔍 Search quotes instantly
- 🎨 Minimal dark UI design
- ⚡ Fast local database (Isar)
- 📱 Responsive layout
