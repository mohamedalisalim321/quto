# 🚀 Quto — Minimal Quotes App

A modern, performance-focused Flutter application for discovering, saving, and searching inspirational quotes.

Built to demonstrate clean architecture, responsive UI design, and efficient local data handling using Isar.

---

## 🎯 Overview

Quto is a lightweight and distraction-free quotes application designed with performance and simplicity in mind.  
The project focuses on clean code structure, scalable architecture, and smooth user experience.

---

## ✨ Features

- 📜 Browse curated quotes
- ⭐ Save and manage favorites
- 🔍 Real-time search
- 🌓 Light & Dark mode
- 🔔 Local notifications
- ⚡ Fast local database (Isar)
- 📱 Fully responsive layout

---

## 🧱 Tech Stack

- **Flutter**
- **Dart**
- **Isar Database**
- **Provider (State Management)**
- **Flutter ScreenUtil**
- **Local Notifications API**

---

## 🏗️ Project Structure
```bash
lib/
├── components/
├── models/
├── pages/
├── databases/
├── themes/
└── main.dart
```

### Architecture Highlights

- Clear separation of concerns  
- Modular folder structure  
- Reusable UI components  
- Local-first data persistence  
- Optimized widget rebuilds
